# Digit Couples

## Description
A number-matching game built with Python and Pygame, featuring multiple difficulty levels, limited health points, and a unique number discovery mechanic.

## Features

- Three difficulty levels:
  - Easy Peasy
  - Oh Ok
  - Oh Lord!

- Randomized number placement
- Memory-based matching gameplay
- Hidden main number discovery mechanic
- Health system
- Win and lose states
- Sound effects
- Interactive menu system

## Requirements
Before running the game, make sure you have the following installed:

- Python 3.x
- Pygame library

## Installation

To install and run Digit Couples, clone the repository and install the required dependency:

```bash
git clone https://github.com/ParsaShabankhah/DigitCouples.git
cd DigitCouples
pip install pygame
python DigitCouples.py
```
